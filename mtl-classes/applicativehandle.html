ApplicativeHandle

ApplicativeHandle[F, E] extends FunctorRaise with the ability to also handle raised errors. It adds the handleWith function, which can be used to recover from errors and therefore allow continuing computations.

The handleWith function has the following signature:

def handleWith[A](fa: F[A])(f: E => F[A]): F[A]

This function is fully equivalent to handleErrorWith from cats.ApplicativeError and in general ApplicativeHandle is fully equivalent to cats.ApplicativeError, but is not a subtype of Applicative and therefore doesn’t cause any ambiguitites.

Let’s look at an example of how to use this function:

import cats._
// import cats._

import cats.data._
// import cats.data._

import cats.implicits._
// import cats.implicits._

import cats.mtl._
// import cats.mtl._

import cats.mtl.implicits._
// import cats.mtl.implicits._

def parseNumber[F[_]: Applicative](in: String)(implicit F: FunctorRaise[F, String]): F[Int] = {
  // this function might raise an error
  if (in.matches("-?[0-9]+")) in.toInt.pure[F]
  else F.raise(show"'$in' could not be parsed as a number")
}
// parseNumber: [F[_]](in: String)(implicit evidence$1: cats.Applicative[F], implicit F: cats.mtl.FunctorRaise[F,String])F[Int]

def notRecovered[F[_]: Applicative](implicit F: FunctorRaise[F, String]): F[Boolean] = {
  parseNumber[F]("foo")
    .map(n => if (n > 5) true else false)
}
// notRecovered: [F[_]](implicit evidence$1: cats.Applicative[F], implicit F: cats.mtl.FunctorRaise[F,String])F[Boolean]

def recovered[F[_]: Applicative](implicit F: ApplicativeHandle[F, String]): F[Boolean] = {
  parseNumber[F]("foo")
    .handle[String](_ => 0) // Recover from error with fallback value
    .map(n => if (n > 5) true else false)
}
// recovered: [F[_]](implicit evidence$1: cats.Applicative[F], implicit F: cats.mtl.ApplicativeHandle[F,String])F[Boolean]

val err = notRecovered[Either[String, ?]]
// err: scala.util.Either[String,Boolean] = Left('foo' could not be parsed as a number)

val result = recovered[Either[String, ?]]
// result: scala.util.Either[String,Boolean] = Right(false)
